## GitHub Pages publishing

This repository now includes `.github/workflows/deploy-pages.yml` to publish the site to GitHub Pages automatically on every push to `main`.

To complete setup in GitHub:
1. Open **Settings → Pages**.
2. Under **Build and deployment**, set **Source** to **GitHub Actions**.
3. Push your built static files to `main` (at repository root), then the workflow will deploy them.
